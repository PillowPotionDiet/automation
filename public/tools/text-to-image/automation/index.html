Google Flow (ImageFX) Browser Automation

This tool automates image generation using your Google account via browser automation.

Configure your settings below, save the config, then run run.bat to start.

Google Login

Enter credentials for auto-login, or use manual login mode.

Image Prompts

Add prompts to generate images for (one per line or use Add button).

    Settings

    Configure automation behavior.

    Instructions

    Step 1: Setup (One Time)

    1. Navigate to the automation folder

    2. Run setup.bat to install Python dependencies

    Step 2: Configure

    1. Fill in your Google credentials above (or use manual login)

    2. Add your image prompts

    3. Click "Save Config" or "Download config.json"

    4. Place config.json in the automation folder

    Step 3: Run

    1. Double-click run.bat

    2. If using 2FA, complete verification when prompted

    3. Wait for images to generate
