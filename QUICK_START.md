## ğŸ’¡ Pro Tips

### For Best Results
1. **Be Specific**: Detailed descriptions = better results
2. **Use Locks**: Enable consistency controls for professional look
3. **Start Small**: Test with 3 scenes before doing 50
4. **Monitor Quota**: Keep eye on API usage (3000 limit)
5. **Save Often**: App auto-saves, but export important logs

### Common Patterns

**Single Location Story**
- Enable Environment Lock
- Describe the location in detail
- Keep lighting consistent

**Character-Focused Story**
- Enable Character Lock
- Upload reference image
- Describe outfit, hair, features

**Multiple Scenes**
- Split naturally by action
- Use transitions for flow
- 3-5 scenes per paragraph works well
